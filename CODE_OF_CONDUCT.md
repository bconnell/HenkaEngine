# Code of Conduct

Henka Engine is an open source project for people interested in practical engine development.

The project should remain respectful, focused, and useful for contributors, testers, and users.

## Expected behavior

1. Be respectful and constructive.
2. Keep discussions focused on the project.
3. Report bugs clearly and honestly.
4. Give feedback in a way that helps the work improve.
5. Respect project scope, maintainership decisions, and licensing.
6. Avoid personal attacks, harassment, insults, or intentionally disruptive behavior.

## Unacceptable behavior

Unacceptable behavior includes:

1. Harassment or threats.
2. Personal attacks.
3. Discriminatory or abusive comments.
4. Spam or repeated off-topic posts.
5. Sharing private information without permission.
6. Pressuring maintainers for unpaid private support, guaranteed timelines, or priority work.

## Scope

This code of conduct applies to project discussions, issues, pull requests, documentation, and other public project spaces.

## Enforcement

Maintainers may moderate, edit, hide, or remove comments, issues, pull requests, or participation that harms the project environment.

This project is early and maintained as time allows. Clear, respectful, scoped contributions are the most useful.
