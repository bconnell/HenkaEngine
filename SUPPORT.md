# Support Henka Engine

Henka Engine is an open source C engine project focused on building practical foundations for 3D, 2D, and 2.5D development.

Sponsorship helps keep the project moving by supporting development time, testing, documentation, examples, packaged builds, and future workspace tooling.

## What sponsorship supports

Sponsorship may help with:

1. Engine development.
2. Runtime testing.
3. Documentation and examples.
4. Sandbox and workspace tooling.
5. Packaged release workflows.
6. Asset pipeline work.
7. Future editor-style workflows.
8. Long-term maintenance.

## What sponsorship does not promise

Sponsorship is voluntary support for the project. It does not purchase:

1. Feature priority.
2. Private support.
3. Guaranteed response times.
4. Ownership of the project.
5. Control over project direction.
6. A separate commercial license.
7. Custom work unless a separate agreement exists.

Project decisions are based on stability, maintainability, usefulness, scope, and the long-term direction of the engine.

## How to sponsor

Use the Sponsor button on the Henka Engine GitHub repository.

## Project direction

Henka Engine is still early. Current work is focused on stable engine foundations, a usable sandbox, viewport tools, transform tools, documentation, packaging, and public examples.

Future work may include stronger asset workflows, improved workspace tools, 2D and 2.5D support, editor-style panels, and better project templates.

## Other ways to help

Sponsorship is appreciated, but it is not the only way to help.

You can also support the project by:

1. Testing builds.
2. Reporting clear bugs.
3. Improving documentation.
4. Suggesting focused improvements.
5. Contributing small, well-scoped pull requests.
6. Sharing the project with developers who may find it useful.
